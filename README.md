# 사용자 카운터 v1.0

## 개요
윈도우용 사용자별 카운터 프로그램입니다. 여러 사용자의 활동을 카운트하고 로그를 관리할 수 있습니다.

## 설치 및 실행

### 필요 사항
- Python 3.6 이상
- tkinter (Python 기본 포함)

### 실행 방법
```bash
python counter_v1.py
```

## 주요 기능

### 1. 사용자 관리
- **사용자 추가**: 이름 입력 후 "추가" 버튼 클릭 (또는 Enter)
- **사용자 삭제**: 리스트에서 선택 후 "삭제" 버튼 클릭

### 2. 카운터 기능
- **+ 증가**: 선택한 사용자의 카운트를 1 증가
- **- 감소**: 잘못 눌렀을 때 카운트를 1 감소 (실수 보정)

### 3. 로그 기록
- 모든 + / - 클릭이 시간순으로 기록됨
- 각 로그에 순번, 시간, 사용자, 동작, 현재 카운트 표시
- 최근 100개 로그만 화면에 표시

### 4. 출력 기능
- "출력 (요약 보기)" 버튼으로 현재 상태 요약
- 형식: `사용자명: N회`
- 클립보드 복사 기능 제공

### 5. 데이터 관리
- **자동 저장**: 모든 작업 후 자동으로 JSON 파일에 저장
- **전체 리셋**: 카운트를 모두 0으로 초기화 (사용자 목록은 유지)
- 데이터 파일: `counter_data.json`

## 데이터 구조
```json
{
  "users": ["사용자1", "사용자2"],
  "counters": {
    "사용자1": 5,
    "사용자2": 3
  },
  "logs": [
    {
      "timestamp": "2025-11-12 10:30:00",
      "user": "사용자1",
      "action": "+",
      "count": 1
    }
  ]
}
```

## 다른 사용자에게 배포하기

### 방법 1: Python 스크립트로 배포
1. `counter_v1.py` 파일을 전달
2. Python 3.6 이상 설치 안내
3. 실행: `python counter_v1.py`

### 방법 2: EXE 파일로 배포 (추천)
PyInstaller 사용:
```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name "카운터" counter_v1.py
```
생성된 `dist/카운터.exe` 파일을 배포

## v1 알려진 문제점 및 개선 사항

### 현재 확인된 문제점
1. 일자별 관리 기능 미구현 (현재는 누적만 가능)
2. 로그 검색 기능 없음
3. 사용자 순서 변경 불가
4. 데이터 백업 기능 없음
5. 로그 내보내기 기능 없음 (현재는 요약만 복사 가능)
6. 키보드 단축키 미지원
7. 대용량 로그 시 성능 저하 가능성

### v2 개선 예정
- 일자별 데이터 관리
- 날짜 필터링 및 검색
- Excel/CSV 내보내기
- 통계 및 차트 기능
- 키보드 단축키 지원
- 사용자 정렬 및 그룹화
