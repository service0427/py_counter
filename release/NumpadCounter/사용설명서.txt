   NumpadCounter - 넘패드 카운터 프로그램
═══════════════════════════════════════════════════════

【 설치 방법 】

1. NumpadCounter.exe 파일을 원하는 위치에 복사하세요.
2. 프로그램을 처음 실행하면 자동으로 counter_data 폴더가 생성됩니다.
3. 설치 완료! 바로 사용하실 수 있습니다.


【 프로그램 시작 】

- NumpadCounter.exe 파일을 더블클릭하여 실행하세요.
- Num Lock이 꺼져있으면 경고 화면이 나타납니다.
  → 화면을 클릭하면 자동으로 Num Lock이 켜집니다.


【 기본 사용법 】

1. 사용자 등록
   - 빈 키(회색으로 표시된 키)를 클릭하세요.
   - 이름을 입력하세요 (2-4글자, 대부분 3글자)
   - 확인을 누르면 등록 완료!

2. 카운트 증가/감소
   - 등록된 키를 클릭하거나 넘패드 키를 누르면 카운트가 증가합니다.
   - 우측 상단의 "+/- 모드 전환" 버튼을 누르면 감소 모드로 전환됩니다.
   - 단축키: - (빼기 키)를 누르면 모드가 전환됩니다.

3. 사용자 수정/삭제
   - 등록된 키를 마우스 우클릭하세요.
   - "수정" 또는 "삭제"를 선택하세요.
   - 수정 시 카운트는 0으로 초기화됩니다.

4. 초기화
   - 좌측 상단의 "초기화" 버튼을 클릭하세요.
   - 모든 카운트가 0으로 초기화됩니다 (이름은 유지됨)


【 프리셋 기능 】

- 화면 상단에 색깔 버튼 3개가 있습니다 (빨강, 파랑, 초록)
- 각 프리셋마다 다른 사용자 그룹을 저장할 수 있습니다.
- 프리셋을 전환하면 해당 프리셋의 사용자와 카운트가 표시됩니다.


【 실시간 로그 】

- 우측에 실시간으로 카운트가 높은 순서대로 표시됩니다.
- 이 영역을 클릭하면 결과가 클립보드에 복사됩니다.
  → 메모장 등에 Ctrl+V로 붙여넣기 가능


【 기타 기능 】

▶ TXT 저장
  - 하단의 "TXT 저장" 버튼을 클릭하세요.
  - 현재 카운트 결과를 텍스트 파일로 저장할 수 있습니다.

▶ 자세히 (일자별 로그)
  - 하단의 "자세히" 버튼을 클릭하세요.
  - 최근 90일간의 로그를 확인할 수 있습니다.
  - 자동으로 90일이 지난 로그는 삭제됩니다.


【 자동 저장 】

- 프로그램은 모든 데이터를 자동으로 저장합니다.
- 프로그램을 종료해도 데이터가 보존됩니다.
- 날짜가 바뀌면 자동으로 히스토리에 저장되고 카운트가 초기화됩니다.


【 데이터 백업 】

프로그램이 있는 폴더의 counter_data 폴더를 백업하세요:
- presets.json: 프리셋 및 사용자 데이터
- counter_data.json: 현재 카운트 및 로그
- history 폴더: 일자별 히스토리


【 문제 해결 】

Q. 프로그램이 실행되지 않아요.
A. Windows Defender나 백신 프로그램에서 차단될 수 있습니다.
   예외 처리를 추가해주세요.

Q. 넘패드 키가 안 눌려요.
A. Num Lock이 켜져있는지 확인하세요.
   화면에 경고가 뜨면 클릭하여 Num Lock을 켜세요.

Q. 데이터가 사라졌어요.
A. counter_data 폴더를 확인하세요.
   백업이 있다면 복원할 수 있습니다.
